# Apple 图片处理小工具

一个基于 **HTML + JavaScript** 的轻量化工具，用于处理 **Apple Live Photo (LIVP)** 和 **HEIC** 格式的图片。  
支持批量转换、预览和下载，方便在电脑端快速处理 Apple 专有格式。

---

## ✨ 功能特点

- **格式支持**
  - 从 `.livp` 文件中提取图片
  - 将 `.heic` 图片转换为 **JPEG / PNG**
- **批量处理**
  - 支持拖拽 / 选择多个文件
  - 多次上传会自动分组显示（第 1 次转换、第 2 次转换…）
- **下载方式**
  - 每次转换结果支持单独打包下载
  - 所有结果支持一键打包下载
- **可视化进度**
  - 单次转换进度条（已处理 / 总数）
  - 每张图片支持缩略图预览 & 点击单张下载
- **现代化界面**
  - 自适应布局，简洁美观
  - 输出格式可切换（JPEG / PNG）
